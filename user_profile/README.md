# User Profile App

The **`user_profile`** app is a core part of the School Management System.  
It handles the **creation and management of all users**, including **students, teachers, and administrators**.  

This app is responsible for storing **personal information** of users and implementing **authentication logic**, such as login, logout, and role-based redirection to the appropriate interface.

---

## Features
- Custom user model with roles: Student, Teacher, Admin
- Student profiles with guardian information
- Teacher profiles
- Role-based authentication and permissions
- Login and logout functionality
- Direct users to the correct interface based on their role
