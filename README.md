# School Management System

A basic school management system for a school from Kindergarten to JHS, built with **Django** and **Django REST Framework**.  
The system aims to digitize core school operations in a simple and scalable way.

---

## Features
- Custom user model with roles: Student, Teacher, Admin
- Student profiles with guardian information
- Teacher profiles
- Role-based authentication and permissions
- Management of classes and subjects
- Academic year and term tracking
- Attendance tracking for students and teachers
- Grading and report cards
- Internal communication system (announcements, messaging)
- Financial management (fees, payments)
- Reports and analytics
- System settings and administration

---

## Technologies
- Python 3.x
- Django
- Django REST Framework
- Git & GitHub
